CORDIC (COordinate Rotation DIgital Computer) is a simple and efficient algorithm to calculate trigonometric functions.

How to use the website ?

  1. Choose an angle for which the trigonometric results want to be known (it must be beetween 0 and 90 degrees);
  2. Choose a number of iteration for the algorithm (it must be beetween 1 and 1000 cycles);
  3. Press the "Run" button;
  4. The results are shown in the table
